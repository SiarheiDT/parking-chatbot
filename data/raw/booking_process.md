# Parking Reservation Process

## Overview

Users can request parking reservations through the chatbot system.

## Required Information

To create a reservation request, the following data must be provided:

- First name
- Last name
- Car plate number
- Reservation start date and time
- Reservation end date and time

## Reservation Flow

1. User initiates a reservation request
2. Chatbot collects required information step-by-step
3. The request is stored in the system
4. A parking administrator reviews the request
5. The reservation is confirmed or rejected

## Important Notes

- A reservation is not guaranteed until confirmed by an administrator
- Users must provide accurate information
- Incorrect or incomplete data may result in rejection

## Changes and Cancellation

- Reservations can be modified or cancelled before the start time
- Late changes may not be accepted

## Support

If you experience issues with booking, please contact support via email or phone.
